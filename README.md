# Кластеризация данных о физической активности

## Описание проекта

В современном мире носимые устройства, такие как фитнес-браслеты, умные часы и различные сенсоры, играют ключевую роль в мониторинге физической активности человека. Эти данные используются в медицинских, спортивных и навигационных приложениях, помогая отслеживать состояние здоровья, уровень нагрузки, а также выявлять потенциальные угрозы безопасности.

В данной задаче предлагается обработка и кластеризация данных, полученных с трех инерциальных измерительных модулей (IMU) и пульсометра. Это позволит автоматически распознавать тип выполняемой физической активности (ходьба, бег, приседания и т. д.), что может быть применено для персонализированного фитнес-трекинга, медицинской диагностики и анализа поведения пользователей.

Исходные данные включают как открытые датасеты, так и собственноручно собранные записи.

## Что было сделано

Выполненные этапы работы

В ходе выполнения проекта были реализованы следующие шаги:

1) Проведен анализ структуры датасета и признакового пространства.

2) Проверены и обработаны пропущенные значения (замена медианой).

3) Удалены неинформативные признаки (timestamp, subject_id).

4) Выполнено снижение размерности признаков путем вычисления норм:

акселерометра
гироскопа

магнитометра

ориентации сенсоров


5) Объединены высококоррелированные признаки.

6) Выполнено масштабирование данных с помощью StandardScaler.

7) Применен метод главных компонент (PCA).

Выбрано 12 компонент, объясняющих 95% дисперсии данных.

8) Проведена кластеризация алгоритмами:

- KMeans

- DBSCAN

- Agglomerative Clustering

Подбор числа кластеров выполнен с использованием метода локтя и silhouette score.

Сформирован submission-файл в требуемом формате для оценки результата.


## Итог

В результате работы выполнена предобработка данных, снижена размерность признакового пространства и проведена кластеризация наблюдений. Алгоритм KMeans показал наилучшие результаты среди рассмотренных моделей и был выбран в качестве итогового решения.
